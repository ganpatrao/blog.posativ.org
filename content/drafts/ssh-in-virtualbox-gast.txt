SSH in ein VirtualBox-Gast

#filters: [reST]

Als ich einmal Gentoo Linux testen wollte, wurde im `Gentoo Buch von Gunnar Wrobel
<http://www.wuala.com/posativ/Documents/Creative%20Commons/Gentoo%20Linux%20-%20Gunnar%20Wrobel%20%5BOpenSourcePress%5D.pdf>`_
empfohlen, sich via SSH einzuloggen (den Gedanken konnte ich teilen).

Nun hatte ich zwar NAT als Netzwerktyp aktiviert und auch eine IP von meinem physikalischem Router erhalten, konnte aber dennoch
keine Verbindung aufbauen. *Nicht einmal pingen konnte ich*

Die Lösung ist:

.. raw:: html
        <pre class="code">
        $ VBoxManage setextradata <Host Name> "VBoxInternal/Devices/pcnet/0/LUN#0/Config/ssh/GuestPort" 22
        $ VBoxManage setextradata <Host Name> "VBoxInternal/Devices/pcnet/0/LUN#0/Config/ssh/HostPort" 2222
        $ VBoxManage setextradata <Host Name> "VBoxInternal/Devices/pcnet/0/LUN#0/Config/ssh/Protocol" TCP
        </pre>

Danach muss die virtuelle Maschine geschlossen werden und neu gebootet werden. Das war's.
Nun kann man sich via ``ssh -l <user> -p 2222 localhost`` einloggen. *(nachdem natürlich vorher
der sshd gestartet wurde: /etc/sshd start)*

via `Howto Access via ssh a Virtualbox Guest machine <http://mydebian.blogdns.org/?p=148>`
