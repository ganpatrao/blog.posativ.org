- [The Hitchhiker’s Guide to  Python!][0] – etwas hübscher
  und realitätsbezogener.
- [Pinry][1] – ist ein privates Flickr in Django geschrieben und sieht echt
  gut aus.
- [ack][2] – *better than grep*
- [lolcommits][3] – je Git Commit ein Foto mit der Webcam. Geil!
- [Plumbum][4] – die Shell in Python, aber diesmal richtig™. Ein [pbs][5]++.
- [0bin][6] – client-seitig verschlüsselter Pastebin. Vielleicht eine
  Alternative zum kürzlich geschlossenen `paste.pocoo.org`.
- [Celery][7] – weiß gar nicht, warum ich das aufgeschrieben hatte. Aber
  *Distributed Task Queue* in Python ist immer nützlich. Supported neben Fork
  auch Eventlets.
- [ansi2html][8] – bunte Bash nach HTML, [bei mir][9] sogar inline.
- [formd][10] – Verwandelt Markdown inline link style in reference style.
  Gehört nach `/usr/local/bin/formd`.
- [git-annex][11] – ein verteiltes Git für seine Daten ohne das Git überläuft.
  Funktioniert für Nomaden wie für Archivisten und unterstützt auch S3 (+
  encryption) und [bup][12]. Hätte ich keine Time-Machine, wäre das mein Tool
  der Wahl.
