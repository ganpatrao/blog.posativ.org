# static blog/site compiler written in Python

- [rstblog](https://github.com/mitsuhiko/rstblog) – reStructuredText und mehr nicht.
- [Liquid Luck](https://github.com/lepture/liquidluck)
- [Volt](https://github.com/bow/volt) – ein weiterer static blog compiler in Python, der genau nichts besser macht.
- [Tinkerer](http://tinkerer.bitbucket.org/) – ein weiterer static blog compiler in Python, der aber auf dem Dokumentationstool Sphinx basiert und [viele hübsche Directiven für reStructuredText](http://sphinx.pocoo.org/markup/index.html) mitbringt.
- [Nikola](http://nikola.ralsina.com.ar/) – ist auch ein statische Seiten Generator mit Blog-Hintergrund und featured als Besonderheit i18n.
- [StrangeCase](http://pypi.python.org/pypi/StrangeCase/v3.4.0) – ein yet another static site compiler written as CamelCase, keine erwähnenswerten Features.
- [Alva](http://lateral.netmanagers.com.ar/weblog/posts/alva.html) ist das Gegenstück zu Nikola und lässt einen Blogeintrage im Browser verfassen, die dann statisch gerendert werden.

## Techniques

- [HTML5 Technical Article Beautifier](http://lobstertech.com/lobsterblog.html) – Syntax Highlighting, LaTeX, Matplotlib, GNUPlot, GraphViz und Inline Python
